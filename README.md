# Harsha Balla | Cybersecurity Portfolio

This is my personal cybersecurity portfolio website. I built it while learning more about web development and cybersecurity, and to have something I can show when applying for internships.

I used plain HTML, CSS, and JavaScript so I could understand how everything works and applied WCAG 2.1 guidelines (Level AA).

---

## What this project is

This is a simple portfolio site that includes:

- My skills in cybersecurity and programming  
- Some of my experience (internships, projects, coursework)  
- A few personal projects I’ve worked on  
- Certifications I’ve earned  
- A contact section with my links

## Features

### What I focused on
- Vanilla JavaScript (no frameworks)
- Component-based structure
- Clean layout and simple design
- Mobile-friendly styling
- Basic animations and interactions

### Design
I went with a cyber / InGen-inspired theme just because I thought it looked cool and fit the cybersecurity vibe.

- Dark background with amber/green accents  
- Monospace + modern fonts  
- Simple hover and scroll animations

## What I learned

This project helped me understand:

- How to structure a frontend project without frameworks  
- How DOM manipulation and component loading works  
- CSS variables and reusable styling systems  
- How to organize a larger static website  
- Basic UI/UX decisions for readability and layout
